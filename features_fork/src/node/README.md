# Node.js (via nvm)

Installs Node.js, nvm, and needed dependencies.

Fork from
[ghcr.io/devcontainers/features/node:1](https://github.com/devcontainers/features/tree/main/src/node),
and remove yarn.

## Example Usage

```json
"features": {
  "ghcr.io/ddosakura/devcontainer-playground/node:0": {
    "version": "latest"
  }
}
```
